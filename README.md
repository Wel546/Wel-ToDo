📝 TodoLove — Lista de Tarefas Simples 💖

Um projeto simples de Todo List (lista de tarefas) feito com HTML, CSS e JavaScript — ideal para quem está aprendendo desenvolvimento web e quer ter um app funcional para organizar tarefas.

👉 Demo: https://wel546.github.io/todolove/

🚀 Sobre

TodoLove é uma aplicação de lista de tarefas simples, interativa e minimalista, onde você pode:

Adicionar novas tarefas

Marcar tarefas como concluídas

Remover tarefas

Visualizar tarefas salvas

Esse projeto é ótimo para entender como manipular o DOM, usar eventos de JavaScript e persistir dados simples (ex.: em localStorage).

📁 Tecnologias Usadas

🔹 HTML — estrutura da página
🔹 CSS — estilo visual do app
🔹 JavaScript — lógica de adição, conclusão e remoção de tarefas

🛠️ Funcionalidades

✅ Adicionar novas tarefas
✅ Marcar tarefas como concluídas
✅ Excluir tarefas
✅ Interface simples e responsiva

🚀 Como rodar localmente

Clone o repositório

git clone https://github.com/Wel546/todolove.git


Entre na pasta do projeto

cd todolove


Abra no navegador

Basta abrir o arquivo index.html

Ou usar um servidor local como Live Server no VS Code
